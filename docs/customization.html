Customization

Open Appearance > Customize > ISCP Theme Options to change branding, header CTAs, hero content, colors, typography, contact details, social links, footer content and homepage section visibility.

Logo and Colors

Set the logo in Site Identity. Change primary, accent, text, background, surface and border colors in ISCP Theme Options > Colors.

Homepage Sections

Use ISCP Theme Options > Homepage Sections to enable or disable Hero, Trust Strip, Solutions, Services, Features, Industries, Stats, Process, Case Studies, Testimonials, Pricing, FAQ, Blog and Final CTA sections.

Typography

Use system font stack choices only. ISCP does not load external fonts by default.

Contact and Social Links

Use Contact Details and Social Links sections to populate phone, email, address, WhatsApp, Google Maps, LinkedIn, Facebook, Instagram, YouTube, X/Twitter, GitHub, Dribbble and Behance links.

Menus

Create menus in Appearance > Menus and assign them to Primary, Footer, Social and Utility locations.

Footer

Use the Footer section in the Customizer to edit footer description, layout, newsletter visibility and copyright text.

SEO and Performance

Use Performance & Accessibility settings to enable animations, respect reduced motion, and toggle schema or Open Graph output. Built-in metadata is skipped when a major SEO plugin is active.
