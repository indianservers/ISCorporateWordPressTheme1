ISCP Shortcodes

ISCP includes lightweight shortcodes for common business website elements. They are optional, safe to use in the block editor Shortcode block, widgets and classic content areas.

Notes

Dynamic shortcodes use native WordPress queries and ISCP custom post types. If no matching content exists, publish the relevant items from the WordPress admin first.

Shortcodes do not load external scripts and do not depend on a page builder.
