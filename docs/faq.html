FAQ

Does ISCP require a page builder?

No. It uses native WordPress templates, Customizer options, CPTs, CSS and vanilla JavaScript.

Can I use WooCommerce?

Yes. Basic WooCommerce support and light styling are included when WooCommerce is active.

What image sizes are recommended?

Hero: 1400x900. Card: 800x600. Blog: 1200x675. Case Study: 1200x800. Logo: SVG or transparent PNG.

Recommended plugins

An SEO plugin, caching plugin, form plugin and security plugin may be used as needed. They are optional.
