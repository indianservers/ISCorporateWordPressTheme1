Indian Servers Corp Pro Documentation

ISCP is the WordPress website theme prepared for Indian Servers Pvt. Ltd. It presents software development, ERP, CRM, cloud hosting, AI automation, cyber security and managed digital solution capabilities with native WordPress templates, Customizer settings, reusable homepage sections and custom post types.

Requirements

Recommended WordPress version: 6.0 or newer. Required PHP version: 7.4 or newer. Use a fresh WordPress install for final QA before packaging.

Licensing and Assets

The theme is prepared for GPLv2-or-later distribution. Do not bundle premium fonts, stock photos, copied screenshots, copied icons, or unlicensed third-party assets. Replace placeholder support details and screenshot assets before commercial sale.

Update Process

Back up the site, install the updated theme package, review Customizer settings, check menus and homepage sections, then refresh permalinks from Settings > Permalinks.
